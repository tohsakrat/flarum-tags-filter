# Tags Filter

A [Flarum](http://flarum.org) extension. Filter discussion lists by discussion Tags.

## Installation

Install with composer:

```sh
composer require tohsakarat/tags-filter:"*"
```

## Updating

```sh
composer update tohsakarat/tags-filter
php flarum cache:clear
```
